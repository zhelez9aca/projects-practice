# Отчёты в бинарных форматах

В папке размещаются итоговые файлы отчёта по практике.

## Содержимое

- `practice_report_template.docx` — исходный шаблон отчёта из репозитория.
- `report_source.html` — редактируемый исходник текущего черновика отчёта.
- `build_report.sh` — локальная сборка `DOCX` и `PDF` из `report_source.html`.
- `report.docx` — собранный черновой отчёт в формате Word.
- `report.pdf` — собранный черновой отчёт в формате PDF.

## Что ещё может потребоваться перед сдачей

1. Подставить фактические ФИО ответственного по практике и куратора проекта.
2. После публикации репозитория заменить локальные или временные ссылки на финальный GitHub URL.
3. При необходимости открыть `report.docx` в Word и довести оформление под требования кафедры.
