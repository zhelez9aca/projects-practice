## Вариативная часть задания

Выбрана вариативная часть `Практическая реализация технологии`. Тема проекта: **разработка Telegram-бота и статического сайта по документации проекта TrackSwop**.

Что входит в решение:

- Telegram-бот на Python, который отвечает на вопросы по документации TrackSwop;
- локальное веб-приложение с тем же движком ответов для демонстрации логики без Telegram;
- статический сайт на русском языке о проекте TrackSwop: назначение, архитектура, провайдеры, ресурсы;
- Markdown-документация о том, как материалы сайта и бота были собраны по исходным источникам проекта.

## Быстрый старт

### Локальное веб-приложение

```bash
python3 src/main.py web --port 8001
```

После запуска откройте `http://127.0.0.1:8001`.

### Telegram-бот

1. Установите зависимости:

```bash
python3 -m pip install -r requirements.txt
```

2. Создайте `.env` в корне репозитория:

```env
BOT_TOKEN=<telegram_bot_token>
WEBAPP_HOST=127.0.0.1
WEBAPP_PORT=8001
```

Можно начать с шаблона:

```bash
cp .env.example .env
```

3. Запустите бота:

```bash
python3 src/main.py bot
```
