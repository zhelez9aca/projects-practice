# Описание модификации проекта

## Идея модификации

В качестве творческой доработки вокруг основного проекта `TrackSwop` создан отдельный
слой представления в виде:

- Telegram-бота-справочника;
- локального web-симулятора;
- русскоязычного статического сайта.

## Зачем это нужно

Исходный TrackSwop — desktop-приложение, но для учебной практики важно не только показать
код, а ещё и объяснить:

- что делает проект;
- как устроена архитектура;
- какие провайдеры уже есть;
- где лежат ограничения и точки роста.

Бот и сайт закрывают именно эту образовательную задачу.

## Что изменено

1. Собран нормализованный слой контента по исходникам TrackSwop.
2. Реализован Telegram-бот, отвечающий по документации проекта.
3. Добавлен локальный HTTP API `/api/chat` для демонстрации тех же сценариев.
4. Подготовлен статический сайт на русском языке с обязательными разделами практики.

## Результат

Модификация даёт три практических эффекта:

- **прозрачность** — материалы TrackSwop изложены на русском языке;
- **демонстрационность** — бот и сайт быстро показывают суть проекта на защите;
- **проверяемость** — ответы опираются на README и исходники, а не на свободные формулировки.
